# To-Do List App

A simple and responsive To-Do List application built using **HTML**, **CSS**, and **JavaScript**. This app allows users to add tasks, delete individual tasks, clear all tasks, and store tasks in the browser's local storage so they remain available even after refreshing the page.

---

## Features

- Add new tasks
- Delete individual tasks
- Clear all tasks at once
- Save tasks using Local Storage
- Tasks persist after page refresh
- Add tasks using the **Enter** key
- Clean and responsive user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Usage

### Add a Task
- Type a task in the input field.
- Click the **Add** button or press **Enter**.

### Delete a Task
- Click the **X** icon next to the task.

### Clear All Tasks
- Click the **Clear All** button to remove all tasks.

---

## Local Storage

The application uses the browser's Local Storage to save tasks.

- Tasks are automatically saved when added.
- Tasks remain available after refreshing the page.
- Clearing all tasks removes them from Local Storage.
